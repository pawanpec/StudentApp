# StudentApp API

## Dependencies

* Java 8+ (Java 11 recommended)


## Building

```sh
mvn clean install
```

## Running

```sh
nohup mvn spring-boot:run &
```

## Test API

```sh
http://localhost:7070/health-check

```
##Swagger Documentation URL
```bash
http://localhost:7070/swagger-ui.html
```

echo $(date +%Y%m%d)$(date +%H%M%S)
